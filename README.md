# Sweet Treats - Bakery E-Commerce Website

A responsive, modern bakery e-commerce website built with vanilla HTML5, CSS3, and JavaScript. Browse fresh, organic baked goods and manage your shopping cart seamlessly.

## Live Demo
https://abdulsubhan1511.github.io/Sweet-Treats---Bakery-E-Commerce-Website/

## Key Features

- **Responsive Product Catalog** - Browse and filter bakery items with a clean, mobile-friendly interface
- **Shopping Cart System** - Add/remove items, view totals, and proceed to checkout with persistent cart functionality
- **User Authentication** - Simple login system for personalized shopping experience

## Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Responsive design with flexbox, gradients, and animations
- **JavaScript ES6+** - Dynamic product rendering, cart management, and DOM manipulation
- **GitHub Pages** - Free hosting and deployment

## Project Structure

```
├── index.html           # Home page (GitHub Pages entry point)
├── shop.html            # Product catalog page
├── cart.html            # Shopping cart page
├── checkout.html        # Checkout page
├── login.html           # User login page
├── order-success.html   # Order confirmation page
├── products.js          # Product dataset and shared data
├── home.js              # Home page functionality
├── shop.js              # Shop page functionality
├── cart.js              # Cart functionality
├── checkout.js          # Checkout logic
├── login.js             # Login functionality
├── styles.css           # Global responsive styling
└── images/              # Product images directory
```

## Key Features Implementation

### 1. Dynamic Product Rendering
Products are stored in `products.js` as a JavaScript array and dynamically rendered to the DOM, allowing easy updates without modifying HTML files.

### 2. Responsive CSS Design
- Mobile-first approach with flexbox layouts
- Custom CSS variables for consistent theming
- Smooth transitions and hover effects
- Beautiful gradient backgrounds inspired by bakery aesthetics

### 3. Shopping Cart Functionality
- Add/remove items to cart
- Calculate totals dynamically
- Persistent cart using browser storage
- Clean checkout workflow
